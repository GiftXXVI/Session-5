What is an API?

  • API stands for Application Programming Interface.
  • APIs provide a simple interface (set of available methods) to perform complicated tasks.
  • They make it easy for other applications to use advanced and complex functionality.
  • Examples:
    • NVidia CUDA allows developers to run Python code on GPUs(for AI, Blockchain ...).
    • psycopg2 allows us to develop applications that run database queries on postgresql.
